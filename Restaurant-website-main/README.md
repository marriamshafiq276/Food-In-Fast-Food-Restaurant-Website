# Restaurant Website

A responsive restaurant landing page for a fast food brand called **Food In**. The project is built with plain HTML, CSS, and JavaScript, and includes sections for services, about/testimonials, app download promotion, and footer links.

## Features

- Responsive navigation menu for mobile screens
- Fixed navbar with scroll background effect
- Hero section with call-to-action buttons
- Services section with food ordering, delivery, and quality highlights
- About/customer section with food image cards
- App download/contact section
- Footer with social links and support links
- Scroll-to-top button

## Project Structure

```text
Restaurant-website-main/
+-- img/              # Website images and logo
+-- index.html        # Main page markup
+-- index.js          # Navbar, scroll, and mobile menu interactions
+-- style.css         # Styling and responsive layout
+-- README.md         # Project documentation
```

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Remix Icon
- Google Fonts

## How to Run

1. Download or clone this project.
2. Open the project folder.
3. Open `index.html` in your browser.

No build step or package installation is required.

## Customization

- Update page content in `index.html`.
- Change colors, spacing, and responsive styles in `style.css`.
- Replace images inside the `img/` folder.
- Edit menu and scroll behavior in `index.js`.
